# Cocollabs

Cocollabs is designed to improve the developer networking process, faciliating collaboration between processionals and transforming how tech projects are born and built.
Imagine an intelligent developer marketplace that transforms learning into career acceleration, individual potential into group creativity, and code into opportunity.

### Running locally
1. Clone the application

    `git clone https://github.com/mariobd38/cocollabs.git`
2. Install npm on your local editor

    `npm install`

2. Run the app with Vite

    `npm run dev`
